## 📦 Hướng dẫn cài đặt

**Đối với Windows:**
1. Tải file `.exe` ở trên về máy.
2. Bỏ file `.exe` vào một thư mục trống (ví dụ `C:\GrokTool\Tool Grok Upscale.exe`).
3. Kích đúp mở tool lên và sử dụng (Không cần cài đặt rườm rà).

**Đối với macOS:**
1. Tải file `.zip` ở trên về máy và giải nén.
2. Chạy file ứng dụng bên trong để mở Tool.
