## 📦 Cách cài đặt (Windows)

1. **Tải file** `Grok_Tool_1.0.1_Windows.zip` bên trên
2. **Giải nén** ra thư mục tùy ý (ví dụ: `C:\GrokTool\`)
3. **Chạy** `CAI_DAT.bat` — đợi cài xong (cần Python + internet)
4. **Khởi động** `CHAY_TOOL.bat` là dùng được ✅

> ⚠️ Yêu cầu: **Python 3.10+** đã được cài đặt  
> Tải Python tại: https://www.python.org/downloads/
